# 📊 Students Performance Analysis

A comprehensive statistical analysis of 1,000 students' academic performance across math, reading, and writing scores.

---

## 📁 Project Structure

students-performance-analysis/

├── data/

│   └── StudentsPerformance.csv

├── notebooks/

│   └── analysis.ipynb

├── requirements.txt

└── README.md

## 🔍 Analysis Sections

1. **Data Loading & Exploration** — Loading dataset, initial inspection, missing values
2. **Data Cleaning & Preparation** — Outlier removal, filtering, standardization, new features
3. **Descriptive Statistics** — Mean, median, mode, variance, skewness, correlation, regression
4. **Data Visualization** — Line, bar, pie, histogram, box plot, scatter, heatmap
5. **Categorical Data Analysis** — Label encoding, one-hot encoding, group analysis
6. **Statistical Tests** — One-sample, two-sample, paired t-test, chi-square
7. **Distributions & Simulation** — Normal, Poisson, Exponential distributions

## 📊 Dataset

- **Source:** [Kaggle — Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)
- **Rows:** 1,000
- **Features:** gender, race/ethnicity, parental level of education, lunch, test preparation course, math score, reading score, writing score
